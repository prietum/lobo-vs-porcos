# Como rodar?
**Windows**: baixe a release para Windows e rode o executável ```.exe```.  
**Linux**: instale [LÖVE](https://love2d.org/) e rode o executável ```.love```  

# Controles
**WASD**: movimentação  
**I**: socar  
**O**: soprar (requer 50% energia)  
**P**: devorar (requer 100% energia, cura 2 pontos de vida)  
  
## Ferramentas utilizadas:
**Visuais**: Aseprite  
**Música e efeitos de som**: Dn-Famitracker  
**Framework**: LÖVE  
**Editor de código**: Sublime Text
